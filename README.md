##  프로젝트 소개

VisiPick은 컨베이어 위를 흐르는 부품을 **실시간 영상 분석으로 검사·분류**하고, 정해진 레시피에 맞춰 양품을 트레이에 모은 뒤 **AGV가 창고로 자동 운반**하는 스마트 팩토리 셀입니다.

부품 4종(IC · 터미널블록 · 방열판 · 커패시터)을 대상으로, 종류 분류와 핀 불량 검사를 동시에 수행하며, 불량·중복·불확실 부품을 각각 다른 경로로 처리합니다.

**주요 기능**
- 🎥 상단 카메라 YOLO 기반 부품 종류 분류
- 🔬 측면 카메라 핀 상태 검사 (정상 / 불량)
- 🔀 4갈래 분류 (양품 / 중복 / 불확실 / 불량)
- 📦 레시피 기반 양품 트레이 수집
- 🤖 myCobot 트레이 이재
- 🚗 AGV 라인 추종 자동 운반 (RFID 노드 인식)

##  시스템 구조

```
┌─────────────────────── 검사·수집 셀 ───────────────────────┐
│                                                              │

│              ↑              ↑   ↑           ↓                 │
│          스텝모터      상단캠 측면캠    서보 게이트 ×2        │
│                       (YOLO) (핀검사)   불량↓  중복·불확실↓   │
│                                       [불량]  [반환 컨베이어] │
│                                                ↓             │
│                              양품 → 슈트 → [빈트레이 큐]      │
│                                              ↓               │
│                                      myCobot 트레이 이재      │
│                                              ↓               │
└──────────────────────────────────────────────┼─────────────┘
                                                ↓
┌─────────────────────── AGV 운반 루프 ─────────────────────┐
│   N1(적재) → 라인추종 → J(분기·RFID) → 창고 하역 → 복귀     │
│              IR 8채널+PD      미션 기반 라우팅               │
└────────────────────────────────────────────────────────────┘
```

##  동작 흐름

| 단계 | 동작 |
|------|------|
| **1. 투입·검사** | 부품이 메인 컨베이어로 흐름 → IR 센서가 도착 감지 → 카메라 촬영 발화 → 상단(종류 분류)·측면(핀 검사) 동시 검사 |
| **2. 판정·분류** | 검사 결과를 4갈래로 분기 |
| **3. 수집·완성** | 양품은 슈트로 트레이에 수집 → 레시피(4종 각 1개) 충족 시 완성 |
| **4. 이재** | myCobot이 완성 트레이를 AGV에 이재 → 트레이 라인 1칸 인덱싱 |
| **5. 운반** | AGV가 라인 추종으로 주행 → RFID로 창고 분기 결정 → 하역 → 복귀 |

**4갈래 분류 로직**

| 판정 | 처리 | 경로 |
|------|------|------|
| 양품 + 필요 | 통과 → 수집 | 슈트 → 트레이 |
| 중복 (레시피 이미 충족) | 반환 | 게이트1 → 반환 컨베이어 |
| 불확실 (저신뢰 판정) | 반환 → 재검사 | 게이트1 → 반환 컨베이어 |
| 확실한 불량 | 격리 | 게이트2 → 불량 처리 |

##  핵심 설계 사상

**1. 불확실 ≠ 불량**
저신뢰 판정 부품은 폐기하지 않고 반환 라인으로 보내 재검사합니다. 대부분의 저신뢰는 결함이 아니라 '나쁜 각도·가림'이므로, 곧장 버리면 멀쩡한 부품을 false-reject로 잃습니다. 반환 후 자세가 바뀌면 2차 검사에서 회수됩니다.

**2. 경로 물리 분리**
양품 경로와 반환·불량 경로의 컨베이어를 물리적으로 분리해 교차오염을 0으로 만듭니다. 양품 순도를 소프트웨어가 아니라 물리로 보장합니다.

**3. 카메라 vs 센서 역할 분리 (AGV)**
라인 추종은 IR 센서 어레이로, 노드 위치 인식은 RFID로 역할을 나눴습니다. 카메라는 제어 루프를 느리게 만들어 제외하고, 빠르고 조명에 강한 IR로 결정론적 주행을 확보했습니다.

##  하드웨어 구성

### AGV (운반)

| 부품 | 역할 |
|------|------|
| ESP32 | 메인 제어 · 센서·통신 통합 |
| QTR 라인센서 8채널 | 라인 추종 (PD 제어) |
| RFID (RC522) | 노드 위치 인식 · 창고 분기 판별 |
| 메카넘 휠 4륜 | 방향 전환 · 회전 |
| ACEBOTT 스마트카 쉴드 | 4모터 구동 (드라이버 내장) |
| 초음파 센서 (HC-SR04) | 장애물 감지 (15cm 내 정지) |

**운용 방식:** 1·2호기 교대 — 1호기가 적재 후 출발하면, 그 신호로 홈에서 대기하던 2호기가 출발점으로 복귀. 라인 위 항상 1대만 두어 충돌 방지.

### 부품이동 하드웨어 (검사·분류)

| 부품 | 역할 |
|------|------|
| 메인 컨베이어 (스텝모터) | 검사 라인 · 정밀 속도 제어 |
| 서브 컨베이어 ×2 (DC) | 반환 / 불량 처리 |
| 서보 게이트 ×2 (MG90s) | 불량 격리 / 중복·불확실 반환 |
| 적외선 센서 (E18-D80NK) | 검사 트리거 |
| 비상정지 (IDEC HW1B) | 전원 차단(NC) + 제어기 인식(NO) |
